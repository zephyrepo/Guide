P&W Guide

Welcome to the big friendly guide

This project is an effort to compile knowledge from around Orbis into an easily accessible and navigable resource, functioning as both a comprehensive guide and convenient reference for players.

Effort has been made to concisely outline game mechanics and clearly separate generalised advice on how players might best approach the game.

If you identify errors or would like to help improve the guide content, please consider joining the BFG Discord to discuss it.
